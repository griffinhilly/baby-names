# Baby Name Taste Map

A self-contained interactive baby-name discovery tool: search names you like, see their taste cohort on a 2D map, and get recommendations. Built from SSA national name data (1880–2024).

Live: https://griffinhilly.github.io/baby-names/

Single static file (`index.html`); generated by `build_explorer.py`. No tracking; preference data stays in your own browser.
